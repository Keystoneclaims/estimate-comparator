Simple file format

For the cleanest results, upload text-based PDF estimates. CSV/XLSX still work with columns like:

roomcategorydescriptionquantityunitunit pricetotal
KitchenCabinetryDetach and reset base cabinets12LF35.00420.00

Recognized aliases include area/location for room, trade/category, item/description/line item, qty/quantity, uom/unit, price/unit cost, and amount/RCV/total. PDF parsing works best on text-based PDFs exported from estimating software. Scanned image PDFs may need OCR or a custom parser.
