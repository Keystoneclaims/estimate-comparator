Simple file format

For the cleanest results, upload text-based PDF estimates exported from estimating software. CSV/XLSX still work with columns like:

RoomCategoryDescriptionQuantityUnitUnit PriceTotal
KitchenCabinetryDetach and reset base cabinets12LF35.00420.00

Recognized aliases include area/location for room, trade/category, item/description/line item, qty/quantity, uom/unit, price/unit cost, and amount/RCV/total. Scanned image PDFs may need OCR or custom parsing.
